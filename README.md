# Classroom API

Este projeto consiste em uma API para gerenciamento de salas e reservas, estruturada seguindo padrões de arquitetura em camadas para garantir a separação de responsabilidades.

---

## Estrutura do Projeto

* **api** : Controladores e rotas da interface REST.
* **domain** : Modelos de dados e entidades de negócio.
* **repositories** : Camada de persistência de dados (em memória/banco de dados).
* **schemas** : Definições de entrada e saída de dados (validação).
* **services** : Lógica de negócio e regras da aplicação.
* **main.py** : Ponto de entrada do sistema.
